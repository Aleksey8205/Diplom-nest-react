# Cайт-агрегатор поиска и бронирования книг в библиотеках

## Описание проекта

Сайт-агрегатор бронирования книг — это платформа для удобного поиска и бронирования книг в различных библиотеках.

### Функционал:
- Поиск и бронирование книг в библиотеках.
- Управление книгами и библиотеками для менеджера и администратора.
- Авторизация и управление пользователями (клиенты, менеджеры, администраторы).
- Создание пользователей.
- Чат технической поддержки.

## Технология и стек

- Frontend: React, Redux, React Router, Lucide, React-modal.
- Чистый CSS без использования библиотек.
- Backend: Node.js, Nest.js, EventEmmiter, passport-jwt, Json web token, multer(для загрузки файлов), bcrypt,
- База данных: PostgreSQL (с TypeORM ).
- WebSocket (socket.io) для онлайн-чата

Рекомендуемые версии:
- Node.js ≥ 18.0
- React ≥ 18.0
- Vite ≥ 7.0

## Установка и запуск

1. Скачайте проект с GitHub и разместите его в отдельную директорию.
2. Установите все зависимости с помощью  `npm run build`  в каждой папке.

3. Создайте файл `.env` на основе `.env.example` и заполните необходимые параметры (POSTGRES_HOST, POSTGRES_USER, HTTP_PORT и т.д.).

4. Запустите проект с помощью Docker Compose: из корня проекта `docker-compose up --build`.


## Структура проекта

Корневая папка проекта содержит следующие важные файлы и директории:
- `docker-compose.yml` — настройки Docker.
- `README.md` — руководство по проекту.
- `.env.example` — пример заполнения файла `.env`.
- Директория `nest-back` — код  back-end части.
- Директория `front-vite` — код front-end части.

## Тестирование

- Зарегистрируйтесь и войдите в систему.
- Воспользуйтесь поиском книг и забронируйте одну из них.
- Протестируйте работу чата технической поддержки.

## Документация API
Подробное описание API основные эндпоинты включают:

## Для клиента

- Регистрацию и авторизацию пользователей.
- Поиск и бронирование книг.
- Общение в чате.
- просмотр забронированных книг.

## Для администратора

- Добавление пользователей.
- Поиск пользователей.
- подключение к чату созданному пользователем.
- Создание, редактирование книг и библиотек.

## Для менеджера

- Поиск пользователей.
- подключение к чату созданному пользователем.
- Создание, редактирование книг в определеной библиотеке.
